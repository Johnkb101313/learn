## Logical Data structure

logical data structure:
1. Stack - linear
2. Queue - linear
3. Trees - non-linear
4. Graph - non-linear
5. Hash Table - tabular (表形式)

definition: Logical data structure 是一套應用於 physical data structure 的邏輯規則, 他定義了 physical data structure 應如何被操作或使用, 所以實際上它會在應用程式中使用, 實際上它會在 algorithms 中使用
